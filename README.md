# 🧥 Old Money – Fashion Product Page

A fully functional and responsive fashion product page with the following features:

- ❤️ Like items and store them in localStorage
- 🛒 Add items to the cart and view/remove them
- ✅ “Buy Now” navigates to an order form with the selected product name
- 🖼️ Product image and name appear in both liked and cart views
- 💾 All actions persist even after refresh using `localStorage`

---

## 🔗 Live Preview

[Click here to view the live demo](https://oldmoney-fashion.netlify.app/)

## 🚀 Features

- Add to Cart functionality
- Like/unlike items with heart icon toggle
- Cart/Like sections show product name + image
- Remove items from cart and liked lists
- Product stays "Liked" or "Added to Cart" even after refreshing
- Redirect to order page with product name via URL

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome Icons
- `localStorage` API

---

## 🧾 Order Page

- Fills the selected product name from query string
- Designed for easy extension with form libraries like EmailJS or Netlify Forms
